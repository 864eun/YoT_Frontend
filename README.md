# YoT(요티)
이 프로젝트는 **React**를 기반으로 구축된 프론트엔드 애플리케이션입니다.
현재 개발 중 입니다. ^O^

#라이브러리
axios (^1.7.9): HTTP 클라이언트

설치: npm install axios 또는 yarn add axios

react (^19.0.0): UI 구축을 위한 JavaScript 라이브러리

react-dom (^19.0.0): React 애플리케이션의 DOM 렌더링을 위한 패키지

react-router-dom (^7.1.5): React 애플리케이션의 라우팅을 위한 라이브러리

recoil (^0.7.7): React 애플리케이션의 상태 관리 라이브러리

sass (^1.85.0): CSS 전처리기

typescript (^5.7.3): JavaScript의 정적 타입 검사를 위한 언어

vite (^6.1.0): 빠른 프론트엔드 개발을 위한 빌드 도구

#개발 도구
eslint (^9.20.1): JavaScript 코드 품질 관리 도구

@vitejs/plugin-react-swc (^3.8.0): Vite에서 React를 사용하기 위한 플러그인
